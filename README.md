# FlowerClassifier

## Description

An app that detects flowers' species using a custom CoreML model that runs locally. Made for iOS >= 13.

![Simulator Screen Shot - iPhone 14 Pro - 2023-01-11 at 23 48 38](https://user-images.githubusercontent.com/60455369/211965848-d734ebad-a56d-47da-bb70-3f548ce28ad9.png)

## Dependencies Used

- [CoreML](https://developer.apple.com/documentation/coreml)
- [Vision](https://developer.apple.com/documentation/vision/)
- [SDWebImage](https://cocoapods.org/pods/SDWebImage)
- [SwiftyJSON](https://cocoapods.org/pods/SwiftyJSON)
